Instructions for running the app locally

## Prerequisites

- Node.js (v18+)

## Running Locally

1. **Clone the repo:**

   `https://github.com/ivan1dze/tz_yellow-pine.git`

   `cd tz-yellow-pine`

   `cd matrix`

3. Install dependencies:

   `npm install`

## Start the app:

`npm run dev`

Access the app at http://localhost:5173.
